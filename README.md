<h3 align="center">✨ Key Features</h3>

<ul>
  <li>📝 <strong>Immutable Portfolio:</strong> Projects & education stored permanently on-chain</li>
  <li>🖼️ <strong>Hybrid Storage:</strong> Assets stored on IPFS via Pinata</li>
  <li>🔐 <strong>Secure Access:</strong> Manager-only updates using modifiers</li>
  <li>💰 <strong>Donation System:</strong> Direct ETH support via smart contract</li>
  <li>⚡ <strong>Gas Efficient:</strong> Only CIDs stored on-chain</li>
</ul>

<hr/>

<h3 align="center">🏗️ Contract Interface</h3>

<ul>
  <li><code>insertProject()</code> → Add new project</li>
  <li><code>changeProject()</code> → Update project</li>
  <li><code>insertEducation()</code> → Add education</li>
  <li><code>donate()</code> → Send ETH</li>
  <li><code>withdraw()</code> → Manager withdraw funds</li>
  <li><code>changeImageLink()</code> → Update IPFS references</li>
</ul>
